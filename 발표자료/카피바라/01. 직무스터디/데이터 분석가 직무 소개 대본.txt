데이터 분석가 직무 소개 대본

안녕하세요
데이터 분석가 직무 소개에 대한 발표를 담당하게 된 4조 카피바라의 팀장 한주람입니다.

발표하기에 앞서 목차를 설명해드리도록 하겠습니다.
첫 번째로 데이터 분석가의 정의에 대해 살펴보고
두 번재로 핵심 역량을 하드 스킬과 소프트 스킬의 관점에서 살펴보고
세 번째로 업무 프로세스에 대해 설명드리고
네 번째로 채용 공고를 통해 어떤 인재풀을 원하는지 살펴보고
다섯 번째로 관련 자격증에 대해 살펴보고
마지막으로 데이터 분석가의 종합적인 결론을 내어 말씀드리도록 하겠습니다.

우선 데이터 분석가란
ChatGPT는 데이터를 수집하고, 해석해서 의사결정을 도와주는 사람 이라고 했고
위키백과에는 데이터에 대한 기본 지식을 바탕으로 프로세스 혁신 및 마케팅 전략 결정 등의 의사 결정을 지원하는 직무를 수행하는 전문가 라고 하였으며
네이버 백과사전에서는 데이터를 수집, 처리, 분석하는 역할을 수행하며, 이를 통해 조직의 의사 결정과정을 지원하는 사람이라고 하였습니다.
즉 요약하자면 데이터 분석가는 데이터를 기반으로 조직의 의사결정을 지원하는 사람을 뜻합니다.

데이터 분석가의 핵심 역량을 하드 스킬과 소프트 스킬로 알아보도록 하겠습니다.
하드 스킬은 업무 시 필요한 기술적 스킬을 얘기하는데요,
SQL, 통계 지식, Python/R을 활용할 줄 아는 프로그래밍 능력, 시각화 도구 활용 능력, Excel 사용 능력 등이 있습니다.
소프트 스킬은 기술적인 능력 외에 다른 사람들과 소통하는 능력을 뜻하는데요, 논리적 사고력, 비즈니스 이해력, 커뮤니케이션 능력, 문제 해결 능력 등이 있습니다.

업무 방식은 크게 7가지로 나뉘는데요
문제 정의, 데이터 수집, 데이터 정제 및 전처리, 탐색적 데이터 분석(EDA), 통계 분석 및 모델링, 리포트 작성 및 시각화, 인사이트 제안 및 의사결정 지원이 있습니다.

많은 분들이 궁금해 하실 것 같은 임금 현황에 대해 조사해보았습니다.
신입 최소 2800, 최대 4500만원 평균 3,425만원으로 밝혀졌으며 다른 연차의 연봉이 궁금하신 분들께서는 사람인 AI-Lab에 접속하셔서 살펴보시면 좋을 것 같습니다.
그리고 원티드와 커리어리를 통해서도 임금 현황을 살펴보았는데요,
신입의 경우 약 3~3.5천만 정도를 받는다는 것을 알 수 있고 미국 평균 연봉은 약 9.6천만 원 임을 알 수 있습니다.

채용공고를 통해 어떤 자격요건이 있는지 간단히 살펴보겠습니다.
저와 저희 조원들이 살펴봤을 때 대부분의 데이터 분석 신입은 대졸 학사 이상을 요구했으며 통계, 경영, 수학, 컴퓨터공학 등의 전공을 우대하는 것으로 파악되었습니다.
또한 Python, R, SQL 등 데이터 분석 언어 활용 능력이 기본적 바탕이 되어야 하며 Tableau나 Power BI같은 BI 시각화 툴의 활용 능력 또한 요구하였습니다. 그리고 통계적 기초 지식을 요구하는 곳이 대부분이었습니다. 이 외로는 해당 회사가 요구하는 JD에 맞는 도메인 지식을 우대사항으로 요구하였습니다.

데이터 분석가로서 취득이 가능한 자격증에 대해 알아보겠습니다.
ADsP, ADsP의 다음 단계인 ADP, 빅데이터 분석기사, SQLD, Google Data Analytics Certificate 정도가 있습니다.
ADsP, SQLD는 필기 시험만으로 합격이 가능하지만 빅데이터 분석기사는 필기와 실기 모두 존재합니다. 그리고 Google Data Analytics Certificate는 외국 대학 강좌를 모아놓은 Cousera 사이트에서 관련 강의를 이수 시 발급 받을 수 있습니다. 한글 자막을 지원하니 관심있는 분들은 살펴보시면 좋을 것 같습니다.


결론적으로 얘기하면 데이터 분석가는 데이터 리터러시 능력과 문제 정의 능력, SQL, Python, 시각화 도구 등의 툴을 다루는 능력을 모두 갖춰야 하는 직업이라고 볼 수 있습니다.

감사합니다.
